This section documents the standard JavaScript libraries shipped with the Enonic XP platform.

== Introduction

When building Enonic apps you may access a broad set of platform capabilities by requiring these libraries. Each library is pulled into your app as a `build.gradle` dependency and loaded at runtime via `require('/lib/xp/<library>')`.

TIP: In addition to the list available in this documentation, you may also browse the https://developer.enonic.com/jsdoc[JSDocs version].
