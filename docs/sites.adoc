== Usage

Unlike <<webapps#, XP webapps>>, XP sites do not have a default controller, but instead rely on resolving controllers based on the content being requested.

To implement a site, start by placing a site descriptor file in `src/main/resources/site/site.xml` of your project.

== Multiple apps per site
Sites may be powered by multiple apps. The apps may even collaborate on processing and responding to a single request. Similar to webapps, developers can use filters and mappings to take control over URL patterns when needed.
